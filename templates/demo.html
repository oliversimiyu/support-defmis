To add this chat widget to your website, include these scripts before the closing </body> tag:

<script>
  window.DEFMISChat = {
    apiUrl: 'http://localhost:8000',
    customerId: null, // Auto-generated if not provided
    customerName: '', // Leave empty to prompt user
    customerEmail: '' // Leave empty to prompt user
  };
</script>
<script src="http://localhost:8000/static/js/chat-widget.js"></script>

Customer Information Collection: If customerName and customerEmail are empty, the widget will automatically prompt users to enter their details before starting a conversation. This information is saved locally and used for future conversations.
